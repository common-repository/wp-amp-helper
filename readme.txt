AMP helper for WordPress

More information : https://github.com/apricoton/wp-amp-helper

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-amp-helper` directory, or install the plugin through the WordPress plugins screen directly.
1. Add your AMP template (Path: path/to/wordpress/wp-content/themes/YOUR_THEME_DIR/amp/single.php)
1. Activate the plugin through the 'Plugins' screen in WordPress
